## ğŸµ How It Works

### For Artists
1. Upload your raw, unfinished track
2. Give up control â€” let the world have it
3. Watch as others remix and reimagine it
4. Your original stays pinned at the top
5. Earn 60% royalties if it goes commercial

### For Remixers
1. Find songs that speak to you
2. Download, add to it, change anything
3. Upload your version
4. Community votes with likes
5. Top 10 versions are featured
6. Split royalties if yours wins

### Completion
- A song is "Complete" when any version hits 1,000 likes
- Complete songs can be distributed to Spotify, Apple Music, etc.
- Original creator gets 60%, contributors split 40%
- Special case: If original version wins, creator gets 100%
